## 📖 About KhataBot

KhataBot is a full-stack digital credit ledger platform designed for kirana stores and small businesses to manage customer credit transactions efficiently.

### ✨ Features

- Multi-tenant architecture supporting multiple merchants and shops
- Secure merchant authentication and authorization
- Customer credit and balance management
- Transaction history and ledger tracking
- Global customer deduplication across merchants
- Automated SMS payment reminders via Fast2SMS
- Credit limit breach alerts
- Financial data stored using paise integers to ensure accuracy
- RESTful API architecture with FastAPI
- Responsive frontend built with Next.js

### 🛠 Tech Stack

- **Backend:** FastAPI, Python
- **Database:** SQLite
- **Frontend:** Next.js, React
- **Notifications:** Fast2SMS
- **Authentication:** JWT
- **Version Control:** Git & GitHub

### 🎯 Use Case

KhataBot helps small retailers digitize traditional paper-based credit records, reduce payment delays, and maintain accurate customer transaction histories.
